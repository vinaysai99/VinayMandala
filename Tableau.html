Global COVID-19 Overview

This dashboard provides a comprehensive overview of the global COVID-19 pandemic.

  • Worldwide total cases and deaths
  • Top 10 countries with the highest number of cases relative to their population
  • A world map illustrating the COVID-19 case count for each country

Covid Monthly Analysis

This dashboard provides a detailed analysis of the COVID-19 pandemic from 2020 to present:

  • Monthly statistics on cases, deaths, and vaccinations.
  • Data is further segmented into quarters for a comprehensive view.
  • Bar graphs illustrate trends and comparisons over time.

World Vaccination Analysis

This dashboard provides a comprehensive analysis of COVID-19 vaccinations worldwide, utilizing data visualizations including:

  • A world map depicting vaccination rates in each country

World Climate Data Analysis

This dashboard provides an in-depth analysis of world climate data, featuring the following key insights:

  • World map visualization showcasing temperature changes from 1975 to present, highlighting trends and variations across countries.
  • Top 10 countries with the maximum and Minimum temperature change in previous years.
  • Interactive line graphs with filtering options to display temperature trends for specific countries, enabling detailed exploration and comparison.

Comparing Year over Year Purchases

This dashboard focuses on analyzing purchase frequencies using dynamic parameters and visualization techniques:

  • Filtering the dashboard using a parameter.
  • Dynamically adjusting the reference line to highlight the minimum day where the total percentage of orders exceeds the target parameter value.
  • Synchronizing a dual-axis chart—one displaying a line graph and the other a circle graph—centered on the minimum day above the target parameter.
